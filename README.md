# Abhipsa Satpathy Portfolio

Personal portfolio website for Abhipsa Satpathy, focused on psychology,
consumer behavior, user research, marketing psychology, and qualitative
insight work.

## Tech Stack

- React
- TypeScript
- Vite
- Three.js
- CSS

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, usually:

```text
http://127.0.0.1:5173/
```

## Build

```bash
npm run build
```

The production output is generated in `dist/`.

## Deploy

This project is ready for Vercel.

```text
Framework: Vite
Build command: npm run build
Output directory: dist
```
